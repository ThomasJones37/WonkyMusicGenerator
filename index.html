How to
Fill each input with a value in seconds,
press play & relax

 Don't forget to check out the cycle length !
What's happening ?
Each input represents a note. By filling in each input, you're telling each note how often it'll be repeated.
You're controlling the most important parameter of this musical piece !

 Each time you press play, the key changes, the notes change, but they always go together
They're twelve keys and they're chosen at random each time
My recommendations & tips
  • Use a minimum value of 0.5s
  • Don't go to big !
  • Increment in quarters (+- 0.25 / 0.5 / 0.75)
  • Use same values twice or more !
  • All this will help the result stay musical !


Extra tips
  • Divide smaller than a 1/4 second. Try 1.125 or 5.625
  • Try really small increments (ex: 1.50s to 1.501s)
  • Try to make the cycle length longer that mankind and reflect
